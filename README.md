# Amazon Clone

A responsive Amazon Clone built using **HTML, CSS, and JavaScript**. This project recreates the look and feel of Amazon's homepage while incorporating interactive features and a user-friendly interface. It demonstrates front-end development skills, including responsive design, DOM manipulation, and modern UI implementation.

## Features
- Responsive design
- Interactive navigation bar
- Dynamic UI using JavaScript
- Product sections and cards
- Clean and organized code structure

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
